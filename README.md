# homebrew-hymnbeam

Homebrew tap for [HymnBeam](https://github.com/wanmekwi/hymnbeam) — a church song-lyrics projector for macOS.

## Install

```bash
brew tap wanmekwi/hymnbeam
brew install --cask hymnbeam
```

## Uninstall

```bash
brew uninstall --cask hymnbeam
```

To also remove all app data (song library, backgrounds, preferences):

```bash
brew uninstall --zap --cask hymnbeam
```
